Use Shortcode [sst_brands_slider]

== Description ==
Create Post under CPT Brands and use Shortcode in Page , please make sure free version have no customization option so use wide layout to see good result.


== Installation ==
Upload Plugin zip folder from Plugin or via FTP , crate some post under CPT Brands with adding image as featured image and add some text in excerpt textarea, add Shortcode [sst_brands_slider] in page , Publish it and Enjoy.
